# 商品详情页设计

## 目标

在产品管理平台的商品管理模块中增加商品详情页 DEMO。详情页用于展示商品基础资料、计费项、成本折扣和版本记录，不提供新增、编辑、删除、发布等写操作。

## 页面结构

- 入口：商品列表中的商品名称进入详情页。
- URL：`/product-management-platform/commodities/:commodityId`。
- 外层：复用产品管理平台全屏壳、左上角 LOGO、顶部菜单和商品管理左侧导航。
- 详情页头部：显示商品名称、描述、状态、所属产品类型、负责人、更新时间，右上角展示“更新商品”按钮，按钮仅提示功能未接入。
- 详情页 Tab：
  - 商品概览：双列展示基础字段。
  - 计费项：只读列表，上方有计费项类型、付费类型、搜索筛选。
  - 成本与折扣：只读宽表，横向滚动查看。
  - 商品版本记录：只读列表，包含草稿版本。

## Mock 数据

按商品类型生成 DEMO 计费项：

- SaaS：平台基础订阅、席位、模型调用、算力资源包。
- 私部订阅：专属环境、私有算力、数据隔离存储、运维支持。
- 私部买断：平台买断许可、初始化部署、模型适配、年度维保。
- 维保：标准维保、SLA 响应、版本升级服务。
- 人力服务：L1 科学支持、L2 方案专家、CSM 项目治理、交付实施。
- BioMap Agent - SaaS：Agent 基础订阅、工作流执行量、知识库容量、企业集成服务。

成本与折扣表拆出四类毛利率：自由折扣毛利率、一般折扣毛利率、产品折扣毛利率、出厂价折扣毛利率。

## 范围外

- 不实现更新商品功能。
- 不实现计费项新增、编辑、删除。
- 不实现版本发布、回滚、复制等操作。
